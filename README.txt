Penggunaan :
Install Xampp minimum spesification 5.6.19 Jika sudah ketik URL localhost Masuk ke PhpMyAdmin
1. Create Database dengan nama database_profile
2. Upload database_profile.sql ke database_profile
3. Pindahkan folder AppsDataDiri di htdocs xampp
4. Buka Aplikasi dengan URL localhost/AppsDataDiri
5. Lakukan login di login.php Jika gagal, daftar di daftar.php 
6. Untuk melihat data diri berupa teks di profile, (Profile akan muncul jika semua data sudah terinput)
7. Untuk sementara upload gambar & Video support <50-100 MB
8. untuk sementara Fitur Delete dan edit data melalui database

Note :
*Data Video Dan Foto yang ada sekarang hanyalah berupa data dummy sebagai hasil percobaan dalam pembuatan Web Apps
